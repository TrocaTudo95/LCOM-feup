§ vbe_get_mode_info()

int vbe_get_mode_info ( unsigned short  mode,
mmap_t *  map 
)

Returns information on the input VBE mode, including screen dimensions, color depth and VRAM physical address.

Initializes unpacked vbe_mode_info_t structure passed as an address with the information of the input mode, by calling VBE function 0x01 Return VBE Mode Information and unpacking the ModeInfoBlock struct returned by that function.

Parameters
modemode whose information should be returned
mapaddress of mmap_t with address of vbe_mode_info_t structure to be initialized
Returns
0 on success, non-zero otherwise
